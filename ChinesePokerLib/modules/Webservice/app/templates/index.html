Welcome to ChinesePokerTips - where machine learning meets Chinese poker!

Chinese poker is a card game based on poker hand rankings for two to four players. In a round of Chinese poker, each player has to split their 13 cards face down into three sets - a "front" set of three, a "middle" set of five and a "back" set of five. The only restriction of the split is that the sets from front to back have to increase in rank.

This is not a valid split because the middle set is higher in rank than the back set:
Front (Ace-jack-high):
Middle (Queen-Ten-high Flush):
Back (Two Pairs, Kings and Twos):

After all players have split their cards, the hands are revealed and the splits are compared set-by-set to determine the score. The front sets are compared against each other, the middle sets compared against each other and so on. In each comparison, the player with the stronger set gain a point from the player with the weaker set. If sets are equally ranked, no points are exchanged. Depending on the game variant, additional points may be awarded for other reasons such as winning all sets against an opponent or for a special hand like having all flushes.

The ChinesePokerTips game uses the 1-6 scoring method in which three additional points (on top of the three points gained from set-by-set comparison) are awarded to a player for winning all three sets against (called "scooping") an opponent. Three points are deducted from the opponent who was "scooped", leaving him/her with negative six points from the head-to-head. No other bonuses are awarded in the ChinesePokerTips game variant and hence the sum of all player scores is always zero after each round.

To create this web application I wrote a Python engine that can simulate rounds of Chinese Poker. I then trained a gradient boosting regression model via an iterative procedure which takes in a hand of cards split into the three sets and outputs a quantity representing its overall strength. This model is used to power the AI strategy in the game. Details behind the development of the strategy can be found here. In the near future, some of the game engine functionality will be made available via an API for any interested developers to use.
